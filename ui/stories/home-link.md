## HomeLink
`HomeLink` is a span that contains either the text `home` or the `LogoEdition` component.

**See**

- [Uses of HomeLink](https://github.com/hashicorp/vault/search?l=Handlebars&q=HomeLink)
- [HomeLink Source Code](https://github.com/hashicorp/vault/blob/master/ui/app/components/home-link.js)

**Example**

```js
<HomeLink @class="navbar-item splash-page-logo">
 <LogoEdition />
</HomeLink>
```
