## ICon
`ICon` components are glyphs used to indicate important information.


| Param | Type | Default | Description |
| --- | --- | --- | --- |
| [glyph] | <code>String</code> | <code></code> | The glyph type. |

**Example**
  
```js
<ICon @glyph="cancel-square-outline" />
```
